# leoecslite-asteroids
2d Asteroids game created with Unity3d and LeoEcsLite.

## Controls
- **Move Fwd**
    * W [*Keyboard*]
    * Up Arrow [*Keyboard*]
- **PrimaryWeapon** (bullets)
    * Space [*Keyboard*]
    * Left Button [*Mouse*]
- **SecondaryWeapon** (laser)
    * Control [*Keyboard*]
    * Right Button [*Mouse*]
- **Rotation**
    * A / D [*Keyboard*]
    * Left Arrow / Right Arrow [*Keyboard*]
- **Pause**
    * P [*Keyboard*]
    * Middle Button [*Mouse*]
